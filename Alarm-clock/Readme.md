# ⏰ Modern Alarm Clock (Python, Tkinter, Pygame)

A visually appealing alarm clock built using Python, Tkinter, and Pygame. It features a neon digital clock, a modern dark UI, a popup alarm notification, and MP3 sound playback.

## 🚀 Features
- Neon digital clock display
- Modern dark UI theme
- Popup alarm window when alarm triggers
- MP3 alarm sound using pygame
- “Stop Alarm” button closes popup and stops sound
- Smooth threading (UI never freezes)
- Beginner-friendly clean code

## 📦 Requirements
Install pygame:
pip install pygame

Tkinter is included with Python on Windows.

Ensure `alarm.mp3` is in the same folder as `app.py`.

## ▶️ How to Run
1. Open terminal inside the project folder  
2. Run:python app.py
3. Enter alarm time in `HH:MM:SS` format  
4. Click **Set Alarm**  
5. When the clock reaches the set time, a popup appears and the alarm sound plays  
6. Click **Stop Alarm** to close the popup and stop the sound
